# 🌍 AI Translator

A multilingual AI-powered translator built using Python, Streamlit, and Deep Translator.

## 🚀 Live Demo

https://ai-translator-app-iueznhkt35bpdbjlqwjwt7.streamlit.app/
## 📌 Features

- Translate text into multiple languages
- Simple and responsive user interface
- Translation history
- Download translated text
- Character and word counter
- Dark theme UI

## 🛠 Technologies Used

- Python
- Streamlit
- Deep Translator

## 📂 Project Structure

AI_Translator/
│── app.py
│── translator.py
│── requirements.txt
│── README.md

## ▶️ Installation

```bash
git clone https://github.com/Maanika28/AI-Translator-App.git
cd AI-Translator-App
pip install -r requirements.txt
streamlit run app.py
```
